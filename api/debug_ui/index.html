调试台

AI短剧工作流调试控制台

这个页面只做一件事：把 script → character → video → voice → compose 这五个阶段清楚地摆出来，让你能一眼看懂流程、逐步调试、也能一键跑全链路。

怎么使用

推荐顺序

  1. 先填“主题”和“输出目录”。
  2. 如果想一键跑，直接点“运行完整流水线”。
  3. 如果想分步调，先点 script，再按顺序往后走。
  4. 剧本生成后，可以在下方直接编辑 JSON，再继续后面的阶段。
  5. 右侧会持续显示当前状态、环境检查、目录和最新产物。
两种模式

全流程模式适合快速验证；手动分步模式适合定位问题和单独调试阶段。
